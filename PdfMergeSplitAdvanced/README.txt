PDF-Tools PdfMergeSplitAdvanced Sample
======================================

This sample requires not only the 3-Heights(TM) PDF Split Merge API but also the
3-Heights(TM) PDF to Image API. Specifically, the following files from the
3-Heights(TM) PDF to Image API must be on the PATH or in ..\..\..\bin:

- Pdf2ImgNET.dll
- Pdf2ImgNET.xml
- RendererNET.dll
- RendererNET.xml

The following file from the 3-Heights(TM) PDF to Image API must be on the PATH
or in ..\..\..\bin\Win32 or in ..\..\..\bin\x64:

- Pdf2ImgAPI.dll
